## Array as ADT

Here, array as adt reffer to array as abstract datatype and set of operations on data.

#### Data
1. Array Space
2. Size
3. Length (Number of elements)

#### Operations
1. Display()
2. Add(x)/Append(x)
3. Insert(index)
4. Delete(index)
5. Search(x)
6. Get(index)
7. set(index)
8. Max()/Min()
9. Reverse()
10. Shift()/Rotate()
